4 Hackathon Tracks

One Protocol, Four Problems

AgentScope addresses all four Synthesis tracks with a unified onchain architecture.

Agents that Pay

Scoped Spending

Your agent moves money with zero guardrails. No per-transaction limits, no daily caps, no address whitelists. One bug and your vault is drained.

ScopeToken enforces per-tx limits, daily caps, and contract whitelists. All validated onchain before any ETH moves.
Agents that Trust

Decentralized Reputation

Trust flows through centralized registries. One API shutdown, one policy change, and your agent loses access to its entire reputation history.

TrustAnchor builds trust scores from onchain attestations and ERC-8004 identity. No central authority. No single point of failure.
Agents that Cooperate

Trustless Escrow

Agent deals run on handshake agreements. No enforcement, no milestones, no neutral arbiter. Disputes are he-said-she-said.

DealEngine locks ETH in milestone-based escrow. Evidence is hashed onchain. Disputes trigger arbitration -- not arguments.
Agents that Keep Secrets

Key Separation

Agents use your private key. Every action leaks metadata: spending patterns, contacts, behavior. Privacy is an afterthought.

Two-key model: owner defines rules, agent operates with its own key. Self Protocol ZK proofs verify humanity without exposing identity.
Architecture

How It Works

Three roles. Zero shared secrets. Full onchain auditability.

1

Human Sets Rules

Owner mints ERC-1155 scope tokens that define spending limits, allowed contracts, deal caps, and attestation rights.

→
2

Agent Operates

Agent uses its own wallet. It can spend, deal, and attest -- but only within scope token boundaries. The contract rejects everything else.

→
3

Contract Enforces

Every action produces an onchain receipt. To revoke access: burn the scope token. Instant. Irreversible. No middleman.
